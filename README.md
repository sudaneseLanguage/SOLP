# SOLP

Sudanese Open Language Project (SOLP)

An open-source project dedicated to building datasets, dictionaries, speech resources, and AI tools for Sudanese Arabic.

## Project Structure

- datasets/
- dictionary/
- docs/
- research/
- website/

## Goals

- Build high-quality Sudanese Arabic datasets.
- Preserve Sudanese linguistic heritage.
- Support NLP and AI research.
- Create open resources for everyone.
