# SOLP

Sudanese Open Language Project

SOLP is an open-source initiative dedicated to building high-quality linguistic resources for Sudanese Arabic.

## Goals

- Build an open Sudanese Arabic dataset.
- Create dictionaries and lexical resources.
- Collect conversations and proverbs.
- Support AI and Natural Language Processing (NLP).
- Preserve Sudanese linguistic heritage.
